Junior High Next Year Scheduling Checklist

Enter into MyStudent/Maintenance/Scheduling

Scheduling for Next Year

BE SURE IN EACH SCREEN TO CHECK NEXT YEAR - Next year scheduling should be completed by June 25th

  • STEP 1 - COURSE PREPARATION - Student Registration Tab
    • Use the Course Listing Report to check your course information setup.
    • Use the Course Definitions to update your course information setup.
      • add/delete courses set-up links set auto-generate status (optional)
      set course credit set graduation requirements

      Helpful Hints:

        District Course Master contains information about ALL courses that are available to the schools. It defines for each course number the allowable grading, graduation requirement, etc. information.
        Add credit to seminary until you are done scheduling. This will help when running Registration Status Report.

    Link Type Definitions

    • Matched Link Type (M): Year courses split into semesters (Math, English, etc.) keep same teacher
    • L Link Type (L): Semester courses that need pre-requisite course taken first (Art 1 - Art 2)
    • Couplet Link Type (C): Semester courses taken a full year (study hall)

  • STEP 2 - STUDENT PREPARATION
    • Use the Student Next Year List to verify your next year students.
    • Use the Registration Status Report/Class Errors option to check for next year promotion problems.
    • Correct promotion problems (next year school number and grade) using Student Info/Edit Demographics.

  • STEP 3 - ENTER STUDENT COURSE REQUESTS - Course Request Input Tab
    • Use Auto-Generate Courses (optional, but before using Course Request Input) to assign students the courses you’ve selected to auto-generate.
    • Enter student requested courses using Course Request Input screen indicating Required, Elective or Alternate status.

      • Helpful Hints:

        Course Tally Report to get number of students requesting courses & determine sections needed.
        Registration Status Report to get students with not enough/to many credits requested.

  • STEP 4 - PROPOSED TEACHER MASTER - Resource Allocation Tab
    • Update Next Year Teacher List using Teacher program
    • Create your Proposed Teacher Master using your semester numbered courses adding a section for each period the course is taught (Mrs. Jones teaches Eng 7 for 6 periods – add 6 sections for 1st semester & 6 sections for 2nd semester). Optionally, add needed prep and advisory periods. Use groups to combine a teacher’s multiple courses taught into a single period. Don’t forget to include room numbers, max seats, etc.

      • Helpful Hints:

        Teacher Listing to clean up teachers, remove retired teachers, etc.
        Proposed Teacher Master Report to check for proper course set-up. Check terms, seats, room numbers, number of sections.

    BEFORE PROCEEDING TO STEP 5 - RE-CHECK EVERYTHING TO THIS POINT!

  • STEP 5 - TRANSFER COURSE REQUESTS - Course Request Input Tab
    • Transfer Course Requests checking Enable Link Substitutions from the input program to the scheduling program. From this point you will no longer use the Course Request Input program for entering students next year course requests.

  • STEP 6 - RE-ESTABLISH AND EXPLODE LINK DEFINITIONS - Student Registration Tab
    • Run Re-Establish Link Definitions to look for any student course combinations that need link instructions
    • Run Explode Link Definitions to generate link scheduling instructions and courses.

  • STEP 7 - PERIOD RESOURCE ALLOCATION - Resource Allocation Tab
    • Run Period Resource Allocation for section period assignments. Goal 100% of courses assigned a scheduling period.
    • Print Proposed Teacher Master to create your magnet board. Continue making adjustments in the Proposed Teacher Master until ready to Create Master Schedule.

  • STEP 8 - CREATE MASTER SCHEDULE FROM PROPOSED TEACHER MASTER - Resource Allocation Tab
    • Run Create Master Schedule from Proposed to move your proposed schedule over to the master schedule.

      • Helpful Hints:

        Course Tally to check for proper course request link explosion.
        Master Schedule to check for proper course set-up. Check terms, seats, room number, number of sections vs. number of requests.

  • STEP 9 - RUN AUTOMATIC SCHEDULER - Automatic Scheduling Tab
    • Run Automatic Scheduler for scheduling students into classes.

      • Helpful Hints:

      From this point you will need to manipulate your Master Schedule to get your best run. After each run of the Automatic Scheduler, print your Master Schedule and document you load percentage so you have something to back to.
